F.A.Q

常見問題

  • 邀請至伺服器了，然後呢?


    你已經完成87%的步驟了! 只要輸入 / 並點選此機器人的頭貼就可以查看所有指令! 輸入 /giveaway start 開始你的第一個抽獎! 更多指令使用資訊請參考官方文檔!

  • 機器人抽獎有問題怎麼辦?


    先確保機器人傳送訊息的頻道有檢視,發送訊息的權限，如果還有問題請至支援伺服器尋求幫助!

  • 訊息ID是什麼?


    當你在編輯抽獎等等時，需要這個東西，請先至 使用者設定 -> 進階 -> 將開發者模式打開，然後對抽獎訊息按右鍵，並且按下複製ID。如果還有問題請至支援伺服器尋求幫助!
